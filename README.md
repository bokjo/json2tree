# json2tree
JSON to Treeview TkInter

- clone repo  
    `git clone git@github.com:bokjo/json2tree.git`
- cd into project root  
    `cd json2tree`
- create virtualenv  
    `python3 -m venv venv`
- activate virtualenv  
    `source ./venv/bin/activate`
- upgrade pip  
    `pip install --upgrade pip`
- install development dependancies  
    `pip install -r requirements.txt`
